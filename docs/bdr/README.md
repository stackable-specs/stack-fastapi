# Behavior Decision Records — python-uv stack

Behavior contracts for the python-uv stack, governed by
[`docs/specs/practices/bdr.md`](../specs/practices/bdr.md). Each BDR captures a
single capability the system agrees to uphold, with acceptance criteria a
black-box observer can confirm.
